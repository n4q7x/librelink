# librelink
This is a Firefox browser extension that "cleans" the LinkedIn interface, hiding unwanted elements.

Similar to my other repository, it is a tutorial repo which explains how I am making it.

Developing browser extensions can feel slightly more finicky than other types of code because you need good tooling. It should be easy to make a change and then run tests. This repo will begin with explaning how to set that up.
